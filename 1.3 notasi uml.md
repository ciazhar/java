# Notasi UML (Unified Modeling Language)
Notasi UML dapat diartikan sebagai diagram sederhana atau penggambaran ringkas tentang koding kita.
Contoh :
```
  _________________
  | Mahasiswa     |
  |_______________|
  | nim : String  | } Properti/Atribut
  | name : String | }
  |_______________|
  | isiKrs()      | } Methode
  | belajar()     | }
  |_______________|
```
